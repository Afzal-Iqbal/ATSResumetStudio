# **App Name**: ResuMaster

## Core Features:

- Step-by-Step Input: Guide users through filling out personal details, work experience, education, and skills.
- ATS Template Selection: Offer a range of ATS-friendly resume templates optimized for various industries.
- Real-time Optimization Tips: Provide instant feedback on how to improve resume sections based on best practices.
- CV Export: Allow users to download their resumes in multiple formats like PDF and .docx.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey professionalism and trust.
- Background color: Light gray (#F0F4F8) for a clean, uncluttered look.
- Accent color: Muted dark blue (#1E88E5) for highlighting important sections and calls to action.
- Body and headline font: 'Inter', a sans-serif, for a modern and readable appearance.
- Use simple, professional icons to represent different sections and skills.
- Implement a clean, well-organized layout that is easy to navigate and scan.
- Use subtle animations for providing feedback or displaying optimization tips.
